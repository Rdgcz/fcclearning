Exibindo Código em Páginas Web

Como mostrar exemplos de HTML e CSS

Para exibir código HTML ou CSS em uma página web, você precisa converter os caracteres especiais em suas entidades HTML. Isso impede que o navegador interprete o código como parte da página, mostrando-o como texto.

EXEMPLO HTML 
<!-- Exemplo de código HTML -->
<div class="container">
    <h1 lang="pt">Título em Português</h1>
    <p lang="en">This paragraph is in English.</p>
    <p lang="es">Este párrafo está en español.</p>
</div>
EXEMPLO CSS 
/* Exemplo de código CSS */
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

p[lang] {
    border-left: 3px solid #3498db;
    padding-left: 15px;
    margin: 10px 0;
}

p[lang="pt"] {
    color: #2ecc71;
}

p[lang="en"] {
    color: #e74c3c;
}

p[lang="es"] {
    color: #f39c12;
}

Notas Importantes:

  • Use &lt; no lugar de < para o símbolo "menor que"
  • Use &gt; no lugar de > para o símbolo "maior que"
  • Use &amp; no lugar de & para o "e comercial"
  • Para exibir código com formatação, use a tag <pre> para manter a indentação
  • Use a tag <code> para indicar que é um trecho de código
